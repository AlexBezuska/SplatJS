Class: Camera

Camera

new Camera(x, y, width, height)

A basic camera. It's really an Entity, so you can control it in the same way. By changing Entity#x and Entity#y you control what portion of the canvas is viewable. For example, if the Camera is at 50,50, and you draw a rectangle at 200,200, it will appear on the screen at 150,150.
Parameters:
Name Type Description
x number The top-left x coordinate
y number The top-left y coordinate
width number The width on the x-axis. Currently doesn't do anything.
height number The height on the y-axis. Currently doesn't do anything.
