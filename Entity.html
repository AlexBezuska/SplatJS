Class: Entity

Entity

new Entity(x, y, width, height)

The base in-game object, it supports a location and velocity. Entities are boxes, consisting of an x,y coordinate along with a width and height. Entities can have basic collision detection, and can resolve collisions in a basic way.
Parameters:
Name Type Description
x number The top-left x coordinate
y number The top-left y coordinate
width number The width on the x-axis
height number The height on the y-axis
